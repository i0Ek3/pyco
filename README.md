# pyco

> The Python3 version of co(Go).

`pyco` have the same features as [co](https://github.com/i0Ek3/co) but write in Python3.

## Features

- supported Python3
- readable code style
- independence and modelization
- other features same as [co](https://github.com/i0Ek3/co)

## Usage

```Python
$ cd pyco
$ python3 main.py
```
